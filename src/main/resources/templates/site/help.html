1. 사용자 사이트 개요

사용자 사이트는 /site/ 경로에서 동작합니다.

  • /site/ : 메인(대문) 페이지
  • /site/page/{pageCode} : 정적 페이지 뷰
  • /site/board/{boardCode} : 게시판 목록 (boardCode 또는 boardId)
  • /site/board/{boardCode}/post/{postId} : 게시글 상세

2. 메뉴 사용 방법

상단 메뉴는 관리자에서 설정한 메뉴 구조를 기반으로 렌더링됩니다.

  • PC: 상단 바에서 1Depth ~ 하위 메뉴를 호버하면 드롭다운으로 펼쳐집니다.
  • 모바일: 우측 상단 햄버거 버튼을 눌러 전체 메뉴를 아코디언 형태로 볼 수 있습니다.
  • 로그인이 필요한 메뉴는 클릭 시 로그인 페이지로 이동한 뒤, 다시 원래 메뉴로 리다이렉트됩니다.

3. 페이지 타입 메뉴 사용

메뉴 타입이 PAGE인 경우, 연결된 페이지의 pageCode로 이동합니다.

  • 메뉴 클릭 → /site/page/{pageCode}로 이동
  • 관리자 > 페이지 관리에서 pageCode, 제목, 내용을 설정하면 자동으로 메뉴와 연동됩니다.

4. 게시판 템플릿 사용

메뉴 타입이 BOARD인 경우, 선택한 게시판의 목록 화면으로 이동합니다.

  • 목록 URL: /site/board/{boardCode} (boardCode 또는 boardId)
  • 기본 템플릿: 테이블 목록
  • 카드 템플릿: 카드형 목록

게시글 제목을 클릭하면 상세 화면으로 이동하며, 본문과 첨부파일이 함께 표시됩니다.

5. 디자인·템플릿 커스터마이징

사용자 사이트의 스타일과 템플릿은 프로젝트별로 커스터마이징할 수 있습니다.

  • 스타일: src/main/resources/static/site/css/site.css
  • 게시판 템플릿: src/main/resources/templates/site/board/

:root의 CSS 변수를 오버라이드하여 테마 색상과 폰트를 변경할 수 있습니다.
